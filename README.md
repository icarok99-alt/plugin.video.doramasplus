# 🌸 Add-on Doramas Plus para Kodi

## 📌 Descrição

O **Doramas Plus** é um add-on para o **Kodi** dedicado ao universo dos dramas asiáticos. Com ele, você tem acesso a um amplo catálogo de:

* 🇰🇷 **Doramas Coreanos**
* 🇨🇳 **Doramas Chineses**
* 🇯🇵 **Doramas Japoneses**
* 🇹🇭 **Doramas Tailandeses**

O add-on utiliza scrapers atualizados para oferecer links funcionais e uma navegação simples e organizada.

> ✅ **Recomendação:** a melhor forma de instalar e manter o Doramas Plus sempre atualizado é através do **Repositório OnePlay**, que gerencia automaticamente dependências e novas versões.

## 🚀 Instalação (Método Recomendado)

### 🔹 Instalar via Repositório OnePlay

1. Abra o **Kodi**
2. Vá em **Configurações** ⚙️
3. Acesse **Gerenciador de Arquivos**
4. Clique em **Adicionar fonte**
5. Insira a URL do repositório OnePlay:

```
https://oneplayhd.github.io
```

6. Nomeie como **OnePlay** e confirme
7. Volte ao menu principal e vá em **Add-ons**
8. Selecione **Instalar a partir de um arquivo ZIP**
9. Escolha **OnePlay** e instale o repositório
10. Depois, vá em **Instalar a partir do repositório → OnePlay → Add-ons de vídeo → Doramas Plus**

Pronto! 🎉 O Kodi cuidará de todas as dependências automaticamente.

## 🛠️ Instalação Manual (Opcional)

> ⚠️ Este método não é recomendado para iniciantes.

Caso prefira instalar manualmente, será necessário instalar as dependências abaixo **antes** do add-on Doramas Plus:

### Dependências

Instale **todas as dependências abaixo antes** do add-on:

* [**script.module.resolveurl.fork**](https://github.com/oneplayhd/oneplayhd.github.io/raw/refs/heads/master/script.module.resolveurl.fork/script.module.resolveurl.fork-5.1.195.zip)
* [**script.module.cloudrequest**](https://github.com/oneplayhd/oneplayhd.github.io/raw/refs/heads/master/script.module.cloudrequest/script.module.cloudrequest-1.0.2.zip)

Após instalar todas as dependências, instale o add-on:

* [**plugin.video.doramasplus**](https://github.com/oneplayhd/oneplayhd.github.io/raw/refs/heads/master/plugin.video.doramasplus/plugin.video.doramasplus-1.1.8.zip)

A instalação é feita pelo caminho:

**Add-ons → Instalar a partir de um arquivo ZIP**

## ℹ️ Observações Importantes

* Utilize sempre a versão mais recente do Kodi
* Para melhor experiência, mantenha o ResolveURL atualizado
* Em caso de erros, limpe o cache do Kodi e reinicie
* Todos os vídeos são provenientes de sites de terceiros — o add-on não hospeda nenhum conteúdo
